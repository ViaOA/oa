== PRSTAT - process stats
    options: "m"= additonal stats
             "L" = lightweight thread
             "p" = pid list (ex: 
             
    > prstat -mL -p1946 5    (once every 5 seconds, keep running)
    
http://www.scalingbits.com/performance/prstat

== MPSTAT processor stats    
    mpstat -p <pid>  [interval  [count]]
    > mpstat -p 5


== VMSTAT virtual memory stats
     vmstat [ interval [count]]    
     > vmstat 5
     


==  Thread Dump to console
