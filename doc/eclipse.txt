20160502 create multiple module ICE project
    1: create a new repo in stash  (ex: tsac/tslm)
            ex: https://stash.intcx.net/projects/TSAC/repos/tslm/browse

    2: eclipse import git project
    3: get template file for ICE project
            Use OABuilder and set Custom Template to "ICETemplate" directory
                and generate custom template files.                
        -OR-
            copy C:\Users\vvia\git\ICE_stash_template  to new directory name
                change all "tslm" to "new_name" in all of pom.xml files

    4: from Eclipse, create a new Java project to point to the new directory
    5: commit to stash
    6: delete eclipse project
    7: eclipse import/existing maven projects
        enter the git location
            ex: C:\Users\vvia\git\udm
        this will create multiple projects in eclipse (like udm)
    8: eclipse: remove ICEstashname project (not needed)
        
    Other:
        eclipse: copy an existing server/client to run 
        copy LogController from another ICE project
        sibling directory for data
            values.properties
                DataDirectory     = ../tsacdata
            
        Resource.java        
            // ICE Custom
            public static final String APP_UniqueServerName  = "UniqueName"; // set by script (for server), and jettyController (for client)

            view/images icons, splash
